Nordit

This is an accounting application that allows users to track daily expenses and incomes. The application features a user-friendly interface that enables users to sort expenses and incomes by date and amount. Additionally, users can edit each expense and income and include supplementary notes. The dashboard presents a comprehensive overview of the user's financial activity for a specified time period which by default is the current month. This includes total income, expenses, and net income.

To ensure secure data storage and user authentication, I utilized Firebase database and Google Authenticator. The application was built using React, SCSS, and HTML, and leveraged several Javascript libraries, including MomentJS and React-dates.

By developing this application, I honed my skills in frontend development, while also gaining some experience in backend development and database management.

Globaltrain

I had the pleasure of building a website for a logistics company, Globaltrain, using React and Firebase for the admin authentication and database. The website features a landing page section that enables users to easily get in touch with the admin through a functional form. Additionally, it automatically generates a unique identification number for each package, making it easier for both the user and the company to keep track of their deliveries.

One of the standout features of the Globaltrain site is the tracking page, which allows users to search for their packages using the provided tracking number. The page provides users with basic information about their packages, as well as the current location of the goods. To ensure optimal performance and a seamless user experience, I utilized client-side routing with React and only load the necessary information from the database when the client initiates a search.

Overall, I thoroughly enjoyed working on the Globaltrain site and am proud of the fast and efficient application I created.

Future

Future is a weather app to be created using React and the OpenWeatherMap API. It will be a platform that allows users to search for and view weather information for different locations around the world. The app will feature a clean, modern interface that is easy to navigate, with a search bar that allows users to enter a city to retrieve the current weather conditions, as well as a five-day forecast.

The integration with the OpenWeatherMap API will enable the app to retrieve real-time weather data, including temperature, humidity, wind speed, and precipitation, for any location entered by the user. The app will also provide users with the ability to toggle between different units of measurement, such as Celsius and Fahrenheit, to ensure that the weather data is displayed in a format that is most meaningful to them.

I am enthusiastic about this project because it would make it easy for people to plan their day-to-day activities based on the weather forecast and it offers an excellent opportunity to enhance my proficiency in AJAX and API data retrieval.
