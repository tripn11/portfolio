My Projects

Nordit

This app is an accounting app. It allows a user to record day to day money flow, both expenses and incomes. It allows the sorting of these expenses and incomes according to date incurred and amount. It also allows the user to edit each expense and income and even include additional notes. On the dashboard, the user can set a desired time period and see the total income, expense and Net Income. The amazing part is that each user's data is saved using Firebase database and Authentication. The app was built using technologies such as React, SCSS, HTML. Firebase database and its Google Authenticator were used for its backend and lastly Javascript libraries such as MomentJS and React-dates were used.
